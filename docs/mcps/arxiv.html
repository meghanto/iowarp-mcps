ArXiv

Data Processing

Fetch and search research papers from the ArXiv repository. Access scientific literature, abstracts, and metadata for research and analysis purposes.

1.0.0 Version
3 Actions
Jan 15 Updated
Python Language

Installation Methods

Global Installation

uv pip install scientific-mcps

Individual MCP Installation

uv pip install "scientific-mcps[arxiv]"

Direct Server

uv run mcp_arxiv

Claude Desktop Configuration

{ "mcpServers": { "arxiv": { "command": "uv", "args": ["run", "mcp_arxiv"] } } }

Available Actions

  • search_papers
    Search ArXiv papers by keywords, authors, or categories
    Parameters: query, category, max_results, sort_by
  • fetch_paper
    Fetch detailed information about a specific paper
    Parameters: arxiv_id, include_abstract, include_authors
  • get_metadata
    Extract metadata from ArXiv papers
    Parameters: arxiv_id, metadata_fields

Usage Examples

Literature Search

// Search for papers on machine learning search_papers( query: "machine learning quantum computing", category: "quant-ph", max_results: 10, sort_by: "relevance" ) // Fetch specific paper details fetch_paper( arxiv_id: "2301.12345", include_abstract: true, include_authors: true ) // Get paper metadata get_metadata("2301.12345", ["title", "authors", "published", "categories"])

Research Workflow

// Search for recent papers search_papers("computational physics", max_results: 20) // Extract paper data for analysis papers_data = [] for paper in results: metadata = get_metadata(paper.id) papers_data.append(metadata) // Analyze with pandas load_csv(papers_data) describe(dataframe, ["publication_date", "citation_count"])
