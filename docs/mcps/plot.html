Plot

Analysis & Visualization

Create powerful visualizations and plots from CSV data using matplotlib and seaborn. Supports multiple chart types and customization options for scientific data presentation.

1.0.0 Version
4 Actions
Jan 15 Updated
Python Language

Installation Methods

Global Installation

uv pip install scientific-mcps

Individual MCP Installation

uv pip install "scientific-mcps[plot]"

Direct Server

uv run mcp_plot

Claude Desktop Configuration

{ "mcpServers": { "plot": { "command": "uv", "args": ["run", "mcp_plot"] } } }

Available Actions

  • create_plot
    Create various types of plots from CSV data
    Parameters: data_file, plot_type, x_column, y_column, title, labels
  • scatter_plot
    Generate scatter plots with customizable markers and colors
    Parameters: data_file, x_column, y_column, color_column, size_column
  • histogram
    Create histograms and distribution plots
    Parameters: data_file, column, bins, normalize, overlay
  • save_plot
    Save generated plots to various formats
    Parameters: plot_object, filename, format, dpi, transparent

Usage Examples

Basic Plotting

// Create a line plot create_plot( data_file: "experiment_data.csv", plot_type: "line", x_column: "time", y_column: "temperature", title: "Temperature Over Time", labels: {"x": "Time (hours)", "y": "Temperature (¬∞C)"} ) // Generate a scatter plot scatter_plot( data_file: "measurements.csv", x_column: "pressure", y_column: "volume", color_column: "temperature" ) // Create histogram histogram( data_file: "results.csv", column: "reaction_time", bins: 20, normalize: true )

Advanced Visualization Workflow

// Load and process data with pandas load_csv("raw_data.csv") filter_data(dataframe, "quality > 0.8") // Create multiple visualizations create_plot(filtered_data, "scatter", "x", "y", "Filtered Results") histogram(filtered_data, "measurement", bins=30) // Save all plots save_plot(scatter_plot, "scatter_analysis.png", "png", 300) save_plot(histogram, "distribution.pdf", "pdf")

Integration with HPC Workflows

// Submit analysis job via Slurm submit_job( job_script: "#!/bin/bash\n#SBATCH --job-name=visualization\nmcp_plot analyze_results.py", nodes: 1, tasks: 4 ) // Create plots from processed results create_plot("simulation_output.csv", "heatmap", title="Simulation Results") save_plot(plot, "simulation_heatmap.png")
